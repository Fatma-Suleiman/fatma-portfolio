# Fatma Suleiman Portfolio

A personal portfolio website showcasing my projects, skills, and professional background. Designed with a clean, modern interface to highlight my work and make it easy for potential collaborators or employers to get in touch.

## Features

- Responsive design
- Project showcase with live demos and GitHub links
- Interactive mini-games (like TicTacToe)
- Smooth animations and modern UI components

## Tech Stack

- React
- Tailwind CSS
- Component-Based Architecture
- Responsive UI Design
- Interactive Elements with JavaScript
